# Go - Claude Sonnet 4.5 Example

## Prerequisites

- Go 1.24+
- Azure CLI (logged in), or any compatible authentication methods for the [DefaultAzureCredential](https://pkg.go.dev/github.com/Azure/azure-sdk-for-go/sdk/azidentity#DefaultAzureCredential).

## Run

```bash
# Run the example
go run .
```

## Files

- `main.go` - Main Go code with EntraID authentication
- `go.mod` - Go module dependencies
- `go.sum` - checksum file for go dependencies.
