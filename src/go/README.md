# Go - Claude Sonnet 4.5 Example

## Prerequisites

- Go 1.21+
- Azure CLI (logged in)

## Setup

```bash
# Initialize Go modules
go mod download
```

## Run

```bash
# Run the example
go run main.go
```

## Files

- `main.go` - Main Go code with EntraID authentication
- `go.mod` - Go module dependencies
